# 安装

安装示例如下：

```python
# 进入源码
cd cqm
# 创建一个build文件夹
mkdir build
# 进入
cd build
# cmake 生成工程文件(自己可以指定参数)
cmake ../
# 编译 
make (gcc, mingw64, clang等编译器)
打开sln工程编译(MSVC编译器)
```
